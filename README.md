
# Data Engineering Portfolio
This repository contains progressive SQL and data engineering mini-projects focused on healthcare operations, referral workflows, facility optimization, and chronic disease intervention analytics.

## Projects

### Project 1: Facility Workflow & Referral Follow-Up Joins

This project uses SQLite and SQL joins to analyze fictional healthcare facility, patient, referral, and follow-up visit data for CareBridge Health.

Skills practiced:
- Relational database design
- Primary and foreign key relationships
- SQL joins
- LEFT JOIN logic
- Aggregations
- Healthcare workflow analysis
